# QA Engineer Project 85

## Тестируемый проект: Codebattle

Codebattle — это игра с открытым исходным кодом, в которой два программиста соревнуются в решении задач на скорость.

### Основные возможности:
- Решение задач против других игроков или бота
- Выбор уровня сложности и языка программирования
- Просмотр хода игры в реальном времени
- Чат для общения участников
- Запись и повтор завершенных игр
- Рейтинговая система

### Содержимое репозитория:
В данном репозитории хранятся тесты, сценарии и отчеты по тестированию проекта Codebattle.
